HTTP API

Примечание

http://212.113.122.139:8000 в ссылках ниже - это IP+PORT на котором развернут биллинг, так что фактические url адреса могут отличаться, например при развертке отдельного инстанса системы отдельному клиенту или переезде на другой сервер.

Документация на основе Django Rest Framework: http://212.113.122.139:8000

Примечание

Для полноценного доступа к документации необходимо запросить авторизационные данные у разработчика/администратора биллинга

Документация на основе Swagger OpenAPI: http://212.113.122.139:8000/api/docs
